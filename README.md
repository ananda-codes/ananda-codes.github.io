# Ananda Biswas — Personal Website

A multi-page personal/portfolio site (Home, About, Research, CV, Writing) with a dark-mode toggle.

## Files
- `index.html` — redirect into the site (GitHub Pages entry point)
- `index.dc.html`, `About.dc.html`, `Research.dc.html`, `CV.dc.html`, `Writing.dc.html` — the pages
- `Nav.dc.html`, `Footer.dc.html` — shared header/footer
- `support.js`, `image-slot.js` — runtime helpers (keep alongside the pages)

## Deploy to GitHub Pages
1. Create a repo (e.g. `ananda-codes.github.io` for a user site, or any repo for a project site).
2. Commit **all** files in this folder, preserving names.
3. Repo → Settings → Pages → Source: `main` branch, `/ (root)` → Save.
4. Your site publishes at `https://<username>.github.io/` (opens the site via `index.html`).

## Editing
- Text, colors, and links are editable in place.
- Drop your headshot onto the photo slots on Home and About.
- Placeholders (education years, extra experience, publications, blog posts) are marked and ready to fill in.
